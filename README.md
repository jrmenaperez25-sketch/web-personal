# Web Personal / CV Online

Esta carpeta ya incluye una web estatica lista para usar:

- `index.html`: estructura de la pagina
- `style.css`: diseno visual
- `script.js`: contenido editable
- `Curriculum JRMP Castellano 2026.pdf`: CV en PDF enlazado desde la web

## Como personalizarla

Abre `script.js` y sustituye:

- `name`
- `role`
- `summary`
- `highlights`
- `skills`
- `timeline`
- `projects`
- `contacts`

## Como verla

Abre `index.html` en el navegador.

## Publicarla rapido

Opciones sencillas:

- GitHub Pages
- Netlify
- Vercel

Si quieres, en el siguiente paso te la dejo ya rellena con el contenido real de tu curriculum y preparada para publicar.
